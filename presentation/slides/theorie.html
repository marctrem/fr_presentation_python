Map

est une fonction d'ordre supérieur ayant la signature:
map(function, iterable, ...)


#
map(function, iterable, ...)
#

m = map(lambda x: 2**x, [0, 1, 2, 3, 4, 5, 6]) # m: <map at 0x7f2308ab7080>

next(m) # 1
next(m) # 2
next(m) # 4

m = map(lambda x, y: (x, y), [0, 1, 2, 3, 4, 5, 6], [2, 4, 6]) # Éq.: zip

next(m) # (0, 2)
next(m) # (1, 4)
next(m) # (2, 6)
